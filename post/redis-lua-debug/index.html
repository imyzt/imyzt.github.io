Redis LUA debug

2020-06-14

调试命令

redis-cli --ldb --eval loop.lua keysargs1 keysargs2, argv1,argv2

  1. 命令解析

--ldb 即开启调试模式
--eval 指定调试脚本
keysargs1 keysargs2, argv1,argv2 逗号前是KEYS的参数， 之后是ARGV的参数。
s 如果，通过 s 执行下一步

  1. loop.lua
local i=0
while true
do
	i=i+1
	redis.debug(i)
end
return "OK"
