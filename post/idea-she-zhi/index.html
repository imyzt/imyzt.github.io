自动生成serialVersionUID

  1. Ctrl+Alt+S打开设置
  2. 搜索栏搜索 Inspections
  3. 选项栏搜索 Serialization issues
  4. 找到 Serializable class without 'SerialVersionUID', 勾选
  5. 应用

对应的实现了 Serializable 接口的类会报出黄色警告, Alt + Enter 组合键选中 Add SerialVersionUID field 即可生成.

去除所有无用引用

  1. Ctrl+Alt+S打开设置
  2. 勾选图中的 show notification after optimize imports action, 确认退出设置
  3. 选中整个项目右键, 点击 optimize imports, 点击确定即可.
  4. 在类窗口, Ctrl+Alt+O, 可以单独去除该类的无用导入.
