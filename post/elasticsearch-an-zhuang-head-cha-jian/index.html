ElasticSearch 安装head插件

2019-05-25

es的安装较简单, 记录一下安装head插件就行了.

首先下载es-head的代码. 地址在这里. https://github.com/mobz/elasticsearch-head
由于head是一个前端程序, 是有跨域问题的, 需要修改es的配置文件, 增加跨域支持. 修改一下 config/elasticsearch.yml 文件, 在最底部增加两行:

http.cors.enables: true
http.cors.allow-origin: "*"

然后进入es-head的目录, 运行 npm install 安装所需依赖, 最后 npm run start 就可以访问浏览器 9100 端口了.
