IDEA 常用插件

2024-01-14 #IDEA

编码类

  1. Generate All Getter And Setter 如其名
  2. AceJump 快速将光标导航到编辑器中可见的任何位置,ctrl+,然后输入字符，进行高亮定位
  3. GsonFormat 将JSONObject格式的String 解析成实体。
  4. Lombok 针对模型对象精简代码
  5. MybatisCodeHelperPro 通过方法名来生成sql，全自动提示，自动检测

工具类

  1. any-rule 可以搜索正则表达式
  2. CamelCase 变量名转驼峰
  3. Jrebel 快速实现热部署，跳过了构建和部署的过程，可以省去大量的部署用的时间。
  4. jrebel-mybatisplus-extension 热部署中mybatis的扩展，他的功能就是配合 Jrebel 一起使用，修改 mybatis 的 mapper.xml 文件不用重启项目
  5. CodeGlance 显示代码地图插件,进行语法突出显示
  6. RestfulToolKit 可以搜索接口
  7. POJO TO JSON 将pojo类转换为json
  8. Translation 翻译插件

Maven类

  1. Maven Helper 依赖冲突分析利器
  2. Maven Dependency Helper Maven中央仓库搜索

规范类

  1. P3C 编码规范的一个插件，高亮提示违规代码
  2. SonarLint 打开文件的时候自动进行代码审查

提效类

  1. Key Promoter X 在IDEA里面使用鼠标的时候，如果这个鼠标操作是能够用快捷键替代，此插件会进行提示
  2. POJO TO JSON 将pojo类转换为json

装修类

  1. Nyan Progress Bar 漂亮的进度条
  2. Rainbow Brackets 配对括号相同颜色，并且实现选中区域代码高亮的功能
